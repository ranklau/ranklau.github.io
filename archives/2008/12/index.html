IE6 SP1+SSL+AJAX(XMLHttpRequest)=trouble

问题描述

在 IE 6 SP1 访问 https 站点，AJAX 会随机出现一个 winet 的状态码：12152。
网上有不少人遇到了这困惑的状态码。

说明

  • Windows 下的 IE 里 new ActiveXObject(progId) 生成的 XMLHttpRequest 是 winet 的一部分。
  • IE 7以下对 SSL (即 HTTPS) 的支持不够好。
